Agente Automatizado de Reseñas
Descripción del Proyecto

Desarrollo de un agente inteligente utilizando n8n que automatiza la gestión de reseñas de clientes para una empresa del sector turístico, generando respuestas personalizadas y creando informes visuales completos.

Metodologías Utilizadas

Automatización de flujos de trabajo, Integración de APIs, Análisis de datos

Tecnologías

n8n, Google Sheets API, Google Drive API, OpenAI API, Webhooks

Características Principales
  • Respuesta automática a reseñas de clientes con IA
  • Integración con Google Sheets para registro de datos
  • Generación automática de gráficas de satisfacción
  • Análisis de sentimiento de las reseñas
  • Informes periódicos enviados automáticamente

Aplicación Gestión de Pasajeros
Descripción del Proyecto

Aplicación web desarrollada durante el primer curso de DAW para una empresa de turismo. El sistema permite gestionar el conteo y control de pasajeros durante el proceso de embarque, asegurando que todos los viajeros estén correctamente registrados antes de iniciar cada viaje.

Metodologías Utilizadas

Desarrollo desde cero sin frameworks, programación estructurada, gestión manual de bases de datos

Tecnologías

PHP puro, MySQL, HTML5, CSS3

Características Principales
  • Sistema de conteo de pasajeros en tiempo real
  • Registro manual de embarque por viaje
  • Validación de pasajeros contra listados de reserva
  • Control de capacidad del vehículo
  • Histórico de embarques realizados
Valor del Proyecto

Este proyecto demuestra un sólido conocimiento de los fundamentos de la programación web. Al estar desarrollado completamente a mano con PHP puro, sin utilizar frameworks ni arquitectura MVC, evidencia la capacidad de construir soluciones funcionales desde cero y comprender en profundidad cómo funciona cada parte del código.
