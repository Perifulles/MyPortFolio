Passenger Management Application
Project Description

Web application developed during the first year of DAW for a tourism company. The system allows managing passenger counting and control during the boarding process, ensuring all travelers are correctly registered before starting each trip.

Methodologies Used

Development from scratch without frameworks, structured programming, manual database management

Technologies

Pure PHP, MySQL, HTML5, CSS3

Main Features
  • Real-time passenger counting system
  • Manual boarding registration per trip
  • Passenger validation against reservation lists
  • Vehicle capacity control
  • Boarding history log
Project Value

This project demonstrates solid knowledge of web programming fundamentals. Being developed entirely by hand with pure PHP, without using frameworks or MVC architecture, it evidences the ability to build functional solutions from scratch and deeply understand how each part of the code works.
